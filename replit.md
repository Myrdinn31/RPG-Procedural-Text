# Overview

Un projet web interactif simple construit avec Python et Flask. Le projet présente un compteur interactif et des cartes d'information animées.

# System Architecture

## Structure du Projet
```
/
├── main.py           # Application Flask (Backend)
├── templates/
│   └── index.html    # Template HTML (Frontend)
├── static/
│   ├── css/
│   │   └── style.css # Styles CSS
│   └── js/
│   │   └── script.js # Logique JavaScript
└── replit.md         # Documentation
```

## Technologies
- **Backend**: Python 3.11 avec Flask
- **Frontend**: HTML5, CSS3, JavaScript ES6
- **Template Engine**: Jinja2 (intégré à Flask)

# Fonctionnalités
- Serveur backend Python gérant les routes
- Interface utilisateur responsive
- Compteur interactif côté client
